SteamFriendsMeter
================

API service for Steam data to Rainmeter

Author
------
Adam Schodde

Version
-------
1.0.0

Installation
------------
```bash
git clone <repo>
cd /path/to/repo
npm install
#edit settings
vim settings.js
node server
```

Don't forget to edit your Rainmeter Skin .ini file to point to your new local/remote server
